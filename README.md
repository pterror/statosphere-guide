# statosphere-guide

A comprehensive unofficial guide to [Statosphere](https://github.com/Lord-Raven/statosphere), the Chub.ai stage extension by Lord-Raven.

**Guide:** https://pterror.github.io/statosphere-guide/

## About

This is a third-party guide. It is not produced or endorsed by Lord-Raven or the Statosphere project. All credit for Statosphere itself goes to [Lord-Raven](https://github.com/Lord-Raven).

The guide covers variables, functions, classifiers, generators, content rules, the stage lifecycle, templating, classification backends, MCP integration, JSON schema reference, and known gotchas. Every nontrivial claim cites a permalink at commit `e67cd9ffaf1ee63e7b5c7bce11462516f547f5f7`.
